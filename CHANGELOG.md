# Changelog

## 1.0.0 — 2026-06-19
First public release.

- Offline Windows voice-to-text with global Ctrl+Shift+Space hotkey.
- Parakeet TDT v3 ASR + finetuned Llama 3.2 3B corrector + KenLM rescoring.
- One-time first-run model download (~2.6 GB); fully offline afterward.
- System-tray app with launch-on-login.
- CI-built installer via GitHub Releases.
