# Screenshots & demo

Drop visual assets here. They make the project far more compelling to anyone skimming the repo.

## Add the demo GIF (highest impact)

1. Install a screen recorder such as [ScreenToGif](https://www.screentogif.com/) (free).
2. Open Notepad (or any editor) next to the recorder.
3. Record yourself pressing **`Ctrl + Shift + Space`**, speaking a sentence, and watching the
   text appear. Keep it short (5–10 seconds).
4. Save it as **`docs/screenshots/demo.gif`**.
5. In the root [`README.md`](../../README.md), uncomment the demo image line:
   `![VoiceFlow demo](docs/screenshots/demo.gif)` (and remove the "coming soon" note).
